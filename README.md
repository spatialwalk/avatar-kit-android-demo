# AvatarKit Demo Project

This is a demo project for the AvatarKit, showcasing the integration and usage of virtual digital humans.

## Before Running the Demo

Before running this demo, please:
1. Set the `Environment` in `MainActivity.kt` to either `cn` (China) or `intl` (International) based on your region.
2. Get your app id and API key from [Open Platform](https://dash.spatialreal.ai) and set them in `MainActivity.kt`.
