# AvatarKit Demo Project

This is a demo project for the AvatarKit, showcasing the integration and usage of virtual digital humans.
It appears to be an Android project, likely built with Gradle and Kotlin.

## Before Running the Demo

Before running this demo, please set the `Environment` in `MainActivity.kt` to either `cn` (China) or `intl` (International) based on your region.

A list of available test avatars can be found at: [https://docs.spatialreal.ai/overview/test-avatars](https://docs.spatialreal.ai/overview/test-avatars)
